## React + Vite + Pnpm 项目模板

一个具有规范标准和统一编码风格的 React18 + TS 企业级项目搭建模板。

所用技术栈：react18、vite、pnpm、typescript、eslint、prettier、husky、commitlint、commitizen、changelog

### 运行

安装 pnpm（如果已安装则跳过该步骤）

``` bash
npm install -g pnpm
```

进入项目目录，执行

``` bash
pnpm install
```

依赖安装后，可直接运行项目

``` bash
pnpm run dev
```
